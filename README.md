# PKU-Met
PKU-Met is a project with the full name of "PKU MetAssistant", which is a python-based assistant designed for meterologists, oceanographers, and Earth scientists. It features:

   - NCEP1 Viewer
   - To-Do List
   - Wikipedia
   - WolframAlpha
   - Yahoo! Weather
   - MP3 Player
